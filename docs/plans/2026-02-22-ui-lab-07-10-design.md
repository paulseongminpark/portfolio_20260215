## 팔레트 퓨전 계열

> OpenAI 레이아웃 골격 + Anthropic 색상/타이포 이식.
> 두 브랜드의 선택이 서로의 영역에 들어갔을 때 어떤 긴장감이 생기는지 실험.

### Page-07 — Warm Index

| 항목 | 값 |
|------|-----|
| 구조 | 좌 고정 인덱스 + 우측 스크롤 (2-a 구조 계승) |
| 배경 | `#f9f6f0` (Anthropic 크림) |
| 텍스트 | `#2c1810` (Anthropic 갈색) |
| 액센트 | `#c4532a` (코퍼 오렌지) |
| 타이포 | Libre Baskerville (헤딩) + Instrument Sans (인덱스/본문) |
| 핵심 긴장감 | OpenAI식 번호 인덱스(`01 / 02 / 03`)가 오렌지로 물드는 순간 |

**배치 이유:**
- OpenAI 인덱스 구조는 방향감과 위계를 동시에 제공하는 강력한 레이아웃
- 그 골격에 Anthropic 팔레트를 얹으면 "차가운 정밀함"이 "따뜻한 정밀함"으로 전환됨
- Instrument Sans(OpenAI 서체)가 크림 배경에서 갖는 낯선 조화가 실험의 핵심

### Page-08 — Warm Card Grid

| 항목 | 값 |
|------|-----|
| 구조 | 고정 헤더 + 카드 그리드 (2-d 구조 계승) |
| 배경 | `#ffffff` (그리드 영역) + `#f9f6f0` (카드) |
| 텍스트 | `#2c1810` |
| 액센트 | `#c4532a` |
| 타이포 | Libre Baskerville (헤딩) + Instrument Sans (카드 메타) |
| 핵심 긴장감 | 흰 배경 위에 크림 카드 — 미니멀 그리드 안에서 따뜻함이 카드 단위로 등장 |

**배치 이유:**
- 카드 그리드는 OpenAI의 대표 정보 전달 방식 (스캐너빌리티 극대화)
- 카드 자체에 Anthropic 크림을 부여하면 배경과 카드 사이의 온도 차이가 생김
- 오렌지 액센트를 카드 상단 경계선 또는 태그에만 사용 → 절제된 따뜻함

## 제3의 미학 계열

> OpenAI의 절제 + Anthropic의 인간성에서 원칙만 추출. 어느 쪽 브랜드도 아닌 새로운 스타일.

### Page-09 — Quiet Precision

| 항목 | 값 |
|------|-----|
| 구조 | 중앙 정렬, max-width 960px, 카드/이미지 없이 텍스트 + 여백만 |
| 배경 | `#fafaf8` (크림도 흰색도 아닌 경계) |
| 텍스트 | `#111111` |
| 액센트 | `#b5863a` (코퍼도 네온도 아닌 차분한 앰버) |
| 타이포 | Playfair Display (헤딩) + Inter (본문) |
| 핵심 긴장감 | 아무것도 없는데 존재감이 있는 레이아웃 — 여백이 유일한 언어 |

**배치 이유:**
- OpenAI의 "여백이 디자인이다" 원칙 + Anthropic의 "텍스트가 서사를 이끈다" 원칙의 교집합
- Playfair Display: 세리프 계열이지만 Libre Baskerville보다 기하학적 → 절제와 인간성 동시
- 앰버 액센트: 오렌지보다 차분, 네온보다 따뜻 → 두 브랜드 사이 어딘가

### Page-10 — Deep Warm

| 항목 | 값 |
|------|-----|
| 구조 | 섹션별 full-width, 좌정렬, 큰 헤딩 |
| 배경 | `#1c1713` (순블랙이 아닌 웜 다크) |
| 텍스트 | `#f0ebe3` (크림 화이트) |
| 액센트 | `#d4845a` (소프트 테라코타 오렌지) |
| 타이포 | Cormorant Garamond (헤딩) + DM Sans (본문) |
| 핵심 긴장감 | 다크 테마인데 차갑지 않음 — "벽난로 앞의 기술 문서" |

**배치 이유:**
- 기존 04(Vercel 다크)는 순블랙 + 틸 → 차갑고 기계적
- 웜 다크 배경은 다크 테마의 실용성 + Anthropic의 따뜻함을 동시에 달성
- Cormorant Garamond: 클래식 세리프 중 가장 날카롭고 우아함. 어두운 배경에서 빛남
- 테라코타 오렌지: 코퍼 오렌지보다 부드럽게 다크 배경과 어울림
